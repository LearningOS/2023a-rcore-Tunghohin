# 2. 简答题
  1. sv39页表项由物理页号和权限位组成
        * V：有效性
        * R: 可读
        * W: 可写
        * X: 可执行
        * U: U模式是否可访问
        * G: 是否全局
        * A：是否被访问
        * D：是否被修改过

  2. 缺页
        * 缺页异常，非法访问
        * scause显示为缺页相关异常
        * 节省空间
        * 约 10^1 MB
        * 访问时若发生缺页异常，查询该地址是否属于 memory_set 内的合法地址，若是即分配物理空间
        * valid 位为 0
  3. 切换 satp
  4. 属于内核空间的 pte 的 U 位设置为 0
  5. 性能相较高，不用频繁切换页表导致cache失效
  6. 系统调用发生时，任务切换时，中断发生时
  7. 任务切换时
